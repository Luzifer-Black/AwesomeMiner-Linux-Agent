Auto installer package.

Erste stabile Version des AwesomeMiner Linux Agent Installers.

- Auto-Installer
- Auto-Download des Remote Agents
- Auto-Repair-System

- GUI-Installer
- Repair-Konsole
- Vollständiges GitHub Projektgerüst
